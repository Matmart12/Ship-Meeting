Bienvenue sur Ship Meeting, le site qui vous propose de rencontrer des personnes avec les mêmes gouts que vous.

Présentation des différents onglets :

1. Ship-avatar : Pour modifier votre avatar ship meeting. Vous ne pouvez le faire qu'une fois par mois sauf si vous avez un abonnement payant. Il est obligatoire pour allez sur le Ship Chat.
2. Ship-chat : Dès que vous avez créer votre ship-avatar, le ship-chat vous met en relation avec une personne qui a le même ship-avatar. Vous pourrez alors entamer une conversation avec cette personne.
3. Formule d'abonnement : Allez sur cette onglet pour voir les différentes offres d'abonnements et leurs avantages.
4. Paramètres : Pour modifier les informations de votre Ship-compte.
